# Owner-approved Gemini bust PNGs

**Epic 3.5 · Sprint 3.5m-docs-gemini**

This folder holds **owner-approved** bust PNGs only — created in Google Gemini Pro Art mode per [`docs/vision/GEMINI_ART_BRIEF.md`](../../docs/vision/GEMINI_ART_BRIEF.md).

## Rules

- **Owner** drops finished PNGs here after completing the owner checklist in the art brief
- **Composer agents** ingest from this folder only — never draw or redraw portrait art
- Log every file in [`docs/vision/references/ASSET_LICENSES.md`](../../docs/vision/references/ASSET_LICENSES.md) before ingest
- Filename pattern: `bust_teen_{hair}_{tone}_{kit}.png`

## Not allowed here

- Agent-generated SVG sources
- Unapproved drafts or WIP variants
- PNGs with baked card frames, gold rings, or HUD chrome
