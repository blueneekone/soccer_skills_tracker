# MASTER BUILD ROADMAP: NEXUS COMMAND & PLAYER OS
**Architecture Philosophy:** Zero-Liability PII, Exception-Only Administration, Dynamic Multi-Sport Scaling, Uncompromised RPG Gamification, and Deep Emotional Intelligence (EQ) Protections.

---

## PHASE 1: FOUNDATIONAL STABILIZATION & ACCESS CONTROL
**Objective:** Eradicate legacy technical debt, secure the database boundaries, and establish an un-squished, ultra-premium visual baseline.

* **Epic 1: The Liquid Glass & Bento Grid Architecture**
  * Transition all dashboard layouts to a responsive CSS Bento Grid utilizing `clamp()` spacing functions to prevent mobile squishing.
  * Soften components with 24px border radii, multi-layered drop shadows, and performant backdrop blur filters.
  * Enforce strict Role-Based Access Control (RBAC) isolating Super Admins, Directors, Coaches, Parents, and Players at the database query level.
  * Implement Cloud Firestore atomic batch writes and increment operators to guarantee offline synchronization and prevent race conditions during telemetry logging.

---

## PHASE 2: ZERO-LIABILITY ONBOARDING & COMPLIANCE
**Objective:** Automate enterprise club ingestion while keeping data storage and payment processing liability at absolute zero.

* **Epic 2: Checkr Native Embed Integration**
  * Deploy the Checkr Client-Hosted SDK directly inside the compliance portal to handle coach Social Security Numbers and Fair Credit Reporting Act (FCRA) disclosures natively.
  * Configure backend token exchange endpoints paired with Alpha mock keys (`temp_alpha_mock_key_pending_ae_approval`) to bypass vendor gatekeeping during testing.
  * Streamline Director workflows with single-click inline clearance attestations wired directly to user IDs, eliminating detached email search forms entirely.

* **Epic 3: Context-Aware COPPA & Household Engine**
  * Build headless CSV bulk ingestion mapping youth rosters to asynchronous "Placeholder Households," rendering an Exception-Only dashboard for registrars.
  * Establish passwordless Magic Uplinks via SMS/Email leveraging single-use, time-locked invite tokens to prevent forwarding attacks.
  * Integrate native **WebAuthn Biometric Enclave Attestation** (`navigator.credentials.create()`) binding legally mandated COPPA consent directly to on-device hardware biometrics (FaceID/TouchID).
  * Decouple asynchronous notifications using the Firestore Outbox Pattern via official Firebase Extensions (`mail` and `messages` collections).

---

## PHASE 3: MULTI-SPORT INFRASTRUCTURE & RPG ENGINE
**Objective:** Decouple rigid sport schemas and launch an addictive, consequence-driven real-life leveling loop.

* **Epic 4: Dynamic Sport-Agnostic Infrastructure**
  * Initialize dynamic configuration trees (`sports_configs`) allowing the OS to scale instantly to basketball, lacrosse, volleyball, etc., by mapping custom attributes, icons, and colors dynamically.
  * Seed open-source, headless YouTube Shorts and looping SVGs/GIFs curated directly to specific attribute buckets to bypass manual content creation.
  * Implement merged collection queries enabling clubs to upload proprietary drills (`tenant_custom_drills`) that cleanly override base libraries.

* **Epic 5: The RPG Progression Loop**
  * Construct multi-attribute hex trees visualizing physical progression across distinct capabilities (e.g., Pace, Ball Mastery, Vision).
  * Enforce hard seasonal level caps requiring verified pitch-side coach sign-offs to unlock advanced tiers.
  * Program automated **Skill Decay** algorithms that drain fractions of inactive XP after 5 consecutive missed days, triggering urgent re-engagement alerts.
  * Integrate Parent Co-Op mechanics, empowering parents to fund real-world rewards via automated escrow **Bounties** alongside one-tap telemetry boosts.

---

## PHASE 4: THE EMOTIONAL INTELLIGENCE (EQ) PERIMETER
**Objective:** Sever horizontal roster comparisons, protect beginner vulnerability, and build an ecosystem explicitly validating player self-worth.

* **Epic 6: Vertical Comparison & Trajectory Tracking**
  * Deploy **Time-Lapse Memory Capsules** that autonomously surface historical baselines during plateaus to prove personal capability.
  * Implement the **Growth Velocity Index** to heavily celebrate rapid month-over-month learning rates for beginners over raw lifetime volume.

* **Epic 7: Psychological Safety & Scaffolding**
  * Implement an asymmetric UI **Fog of War** masking intermediate/advanced skill trees to prevent cognitive overload for complete beginners.
  * Launch zero-floor **Ignition Quests** providing immediate tactile micro-wins within 60 seconds of application load.
  * Introduce **Grit XP** explicitly rewarding failed attempts on highly complex moves to permanently eliminate the shame of dropping the ball.
  * Establish private, un-monitored Intrinsic Joy tracking to give players complete autonomy over their emotional record.

* **Epic 8: Adult EQ Interceptors**
  * Build intent-based homework triggers allowing coaches to assign single macro-goals (e.g., "+150 Ball Mastery XP") while the backend autonomously serves individualized, scaffolding-safe drills per player.
  * Engineer **"The Car Ride Home" Protocol** firing mandatory push notifications to parents 15 minutes post-match, locking metrics behind an attestation to prioritize emotional safety and providing highly empathetic conversation anchors.
