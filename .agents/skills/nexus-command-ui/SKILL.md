#### 2. THE CONTRAST NUKE & MICRO-TYPOGRAPHY
You are STRICTLY FORBIDDEN from using `rgba()` text opacities or Tailwind opacity modifiers (e.g., `text-white/50`) on dark backgrounds. 
*   **Contrast Scale:** Primary Text: `#FAFAFA`; Secondary: `#D4D4D8`; Tertiary/Icon: `#A1A1AA`.
*   **Typography Engine:** Use **Geist Mono** for all data, telemetry, and numerical readouts with aggressive letter spacing (`tw-tracking-widest`). Use **Switzer** for body copy to leverage its 79% x-height for maximum legibility. Use **Geist Sans** for headers with negative tracking (`tw-tracking-tight`) [cite: 727, 1040].

#### 3. MATERIAL HIERARCHY & Z-DEPTH ZONES
Do not mix depth levels [cite: 1053].
*   **Z0 Canvas:** Absolute black void. Maintains a "Void Density" of >= 40% [cite: 1053].

#### 4. ASYMMETRIC GRID & LAYOUT PHYSICS
