SSTRACKER (Nexus Command) - UX/UI Redesign Roadmap
Master Aesthetic Protocol: "Nuclear Americana Tech Noir"
All interface generation must strictly adhere to the Enterprise SIEM/SOAR observability paradigm, fusing 1950s technology firm aesthetics with subtle cyberpunk. Generic Bootstrap templates, neon casino gamification, and standard purple-gradient SaaS dashboards are strictly forbidden.  

Global Styling Mandates
Bento Grid Topology: All layouts must migrate to a padding-neutral 12-column Bento Grid using fluid clamp() variables (e.g., var(--bento-pad-liquid)).  

60-30-10 Color Taxonomy: 60% Dominant Base (Void Black/Navy Slate), 30% Structural Trim (Structural Grey/Silver for 1px borders), and 10% Action & Telemetry (Action Gold, Data Cyan, Atompunk Amber).  

Z-Depth Architecture: Apply a strict five-tier Z-depth layout (Z0 void canvas, Z1 recessed wells, Z2 matte base panels, Z3 raised holographic layers, Z4 floating chrome).  

Geometry & Typography: Enforce chamfered corner clip-paths exclusively; rounded Material Design pills are forbidden. Utilize Geist Display for hero metrics and Geist Mono for heavily letter-spaced (tracking-widest) interface labels.  

Phase 1: The Player OS (Cinematic Command Deck)
Objective: Replace standard checklists with a high-density gaming Heads-Up Display (HUD) that utilizes intrinsic motivation without predatory mechanics.

[ ] Viewport Focus: Enforce the "singular directive focus" rule by restricting the interface to exactly one Action Gold Call-To-Action (CTA) per viewport.  

[ ] Avatar Studio Assembly: Build the drag-and-drop geometric Avatar Studio based on UPA animation and Rubber Hose styles. Implement programmatic anchor points (e.g., NECK_ANCHOR) for tiered unlocks, such as the 20 Atompunk hairstyles and mechanical eye overlays.  

[ ] Composite Snowflake Skill Tree: Design the central progression track using Data Cyan for the active path and Atompunk Amber for secondary circuit nodes. Implement a "Fog of War" visual mechanic to obscure advanced training nodes.  

[ ] Recessed Telemetry Wells: Place "Scout's Six" radar charts inside Z1 recessed wells to reduce visual noise and simulate a physical instrument cluster.  

Phase 2: The Coach OS (Sideline SIEM / SOAR)
Objective: Strip away gamified elements and deliver a highly auditable, cyber-security-style data dashboard for high-stress sideline environments.

[ ] Tactical War Room Canvas: Integrate the Fabric.js spatial drill designer within a dominant 8-column Bento Grid span (bento-span-8). Design diegetic, tactical terminal brackets around the canvas frame.  

[ ] Drag-and-Drop Calendars: Implement SortableJS visual grids for macrocycles and microcycles, ensuring the UI supports rapid, touch-friendly dragging of training blocks.  

[ ] RAG AI Assistant Chat Interface: Build the conversational AI terminal using Geist Mono typefaces for system outputs, wrapping the chat window in 1px Structural Grey metallic borders.  

[ ] Real-Time Telemetry Grid: Map the active roster status and biometric warnings into a full-width bento-span-12 telemetry grid using strict 150-250ms state-change micro-animations.  

Phase 3: The Parent OS (Co-Op Trust Center)
Objective: Deliver an emotionally intelligent, anxiety-reducing environment focused on privacy, billing transparency, and verified rewards.

[ ] Compliance Gateways: Design the WebAuthn (FaceID/TouchID) and passwordless magic link onboarding flows with calm Navy Slate backgrounds and high negative space.  

[ ] Co-Op Arena & Bounty Terminal: Wrap the CoOpArena and BountyTerminal components inside the 12-column liquid grid. Design the UI for the Tremendous Bounty Escrow system where parents can fund real-world rewards.  

[ ] The Car Ride Home Widget: Style the post-match emotional intelligence alerts as raised Z3 holographic cards with Liquid Glassmorphism 2.0 (backdrop-filter: blur(20px)).  

Phase 4: The Director OS (Revenue Command Shell)
Objective: Construct a macro-level executive console for managing massive multi-tenant operations and financial streams.

[ ] Director's SPA Shell: Build a fixed left-hand sidebar for navigation and a main content area with independent vertical overflow scrolling. Implement seamless opacity transitions (display: none toggles) for view switching without page reloads.  

[ ] Executive Dashboard Ribbon: Map the Monthly Recurring Revenue (MRR) and Active Organizations KPI strip into the primary bento-span-12 row.  

[ ] Restricted Chart Containers: Integrate custom, dark-mode Chart.js components (e.g., Seasonal Engagement Trend lines, Global Seat Distribution doughnut charts). Enforce explicit responsive boundaries (max-width, max-height) to prevent the charts from breaking the fluid grid.  

[ ] God Mode Allocation UI: Design the "Licensing" data table view using HTML/Tailwind progress bars to visually track active seat utilization against purchased capacity.
